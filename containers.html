Fluid

class="container-fluid"

This is a fluid container. When resizing the browser, the text will always be a certain length away from the egde of the container. This one can't be demonstraited as well because of the current containers of this code. But if you get to a small enough breakpoint, you will see ALL the text start to move and rush around themselves, adhearing to get to the edge of the container as best they can.

Fixed-Width

class="container"

This is a fixed-width container. There is still padding on each side, but the container adjusts to media query breakpoints instead of being fluid. It will always keep itself center, but it will also always make sure all the content is within the view of whatever viewport is being used. Assuming it is being used correctly.

A Background Image

class="bgimage"

This is to showcase how not everything has to go into a container. In this case we have a header with the class of "bgimage" outside of our div with the class of "container" If you resize your browser, youll see that the background image will adjust apropriately depending on the size. This also applies to how much content is in the container. Also keep in mind colors, as you can see some of this text can be hard to read.
