Welcome to Bootstrap!

Boy you're going to have fun with this...

Bootstrap is a fairly complex (in my opinion, but do mind I consider myself a junior dev) framework. It can be downloaded from here. It can be pretty easy if you can remember a bunch of new / different html styles, css classes and ID's, and all the likes. It's a sort of "controlling" type of framework. If you know how to follow guidelines it can be very, very beneficial. If you can't, well... I guess this guide can help you!

You can do TONS of different things. Containers, styles, responsive utilities, tables, images, forms, formatting code, columns, using clearfix, button styles, breakpoints. Absolutely anything that you can think of. I try my best to style this to make it easy to understand, follow, and learn from. I'll update it as I feel I have the time / motivation to do so, but I won't make any promises. Sorry lads!
