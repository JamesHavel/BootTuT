Formatting Code

You have:

Inline Code

Use the Code Tag which will make it look like code. So in this case I can paste a random bit of code like so:
.icon { max-width: 150px; margin: 0 auto; display: block; }You might have to style it a bit by yourself though if you want it to look neat.

Control Keys

Use the kbd code to make it look as that example is. Useful if you want someone to use a certain key. Example like so: Use the q key.

Short Script Snippits

A way better way to paste code would be to use the pre tag. Using it will make anything look like so:

.bgimage {
          background-position: center center;
          background-size: cover;
          background-image: url('../img/cat.jpg');
          .container {
            p, h1 {
              color: white;
              text-shadow: 2px 2px black;
              font-weight: bold;
              font-size: 20px;
            }
          }
        }
